# 驾校信息管理系统 java509

## 项目概述

驾校信息管理系统是基于SSM（Spring + SpringMvc + Mybatis）和Vue的前后端分离项目。该项目采用MVC三层体系结构，以JavaEE技术栈开发，通过Maven进行构建管理。系统使用MySQL数据库作为数据存储平台，实现了基于B/S结构的Web系统。整体界面简洁，操作方便。

## 技术栈

- 前端：Vue
- 后端：Spring、SpringMvc、Mybatis
- 数据库：MySQL
- 构建工具：Maven
